# Travel Booking Frontend

## Setup

1. Install deps:
```
npm install
```
2. Create `.env.local` with:
```
VITE_API_BASE=http://localhost:5000/api
```
3. Run dev:
```
npm run dev
```

## Scripts
- `npm run dev` – start Vite dev server
- `npm run build` – production build
- `npm run preview` – preview build
